## 1. –§—É–Ω–¥–∞–º–µ–Ω—Ç–∞–ª—å–Ω–∞—è —Ç–µ–æ—Ä–∏—è –∞–≥–µ–Ω—Ç–æ–≤

### 1.1. –û—Ç Chain –∫ Agent: –≠–≤–æ–ª—é—Ü–∏—è Control Flow

–¢—Ä–∞–¥–∏—Ü–∏–æ–Ω–Ω–æ–µ –ø—Ä–æ–≥—Ä–∞–º–º–∏—Ä–æ–≤–∞–Ω–∏–µ –∏ —Ä–∞–Ω–Ω–∏–µ LLM-–ø—Ä–∏–ª–æ–∂–µ–Ω–∏—è —Å—Ç—Ä–æ–∏–ª–∏—Å—å –Ω–∞ –±–∞–∑–µ –∂–µ—Å—Ç–∫–∏—Ö —Ü–µ–ø–æ—á–µ–∫ (Chains).

- **Chain (–¶–µ–ø–æ—á–∫–∞):**¬†–≠—Ç–æ Directed Acyclic Graph (DAG), –≥–¥–µ —Å—Ç—Ä—É–∫—Ç—É—Ä–∞ –≤—ã–∑–æ–≤–æ–≤ –∂–µ—Å—Ç–∫–æ –∑–∞–¥–∞–Ω–∞ –ø—Ä–æ–≥—Ä–∞–º–º–∏—Å—Ç–æ–º (`Prompt`¬†$\to$¬†`LLM`¬†$\to$¬†`Parser`).
    
    - _–ü—Ä–æ–±–ª–µ–º–∞:_¬†–•—Ä—É–ø–∫–æ—Å—Ç—å. –ï—Å–ª–∏ LLM –≤–µ—Ä–Ω–µ—Ç –Ω–µ–æ–∂–∏–¥–∞–Ω–Ω—ã–π —Ñ–æ—Ä–º–∞—Ç –∏–ª–∏ –ø–æ—Ç—Ä–µ–±—É–µ—Ç —É—Ç–æ—á–Ω–µ–Ω–∏—è, —Ü–µ–ø–æ—á–∫–∞ —É–ø–∞–¥–µ—Ç. –û–Ω–∞ –Ω–µ —É–º–µ–µ—Ç –∞–¥–∞–ø—Ç–∏—Ä–æ–≤–∞—Ç—å—Å—è.
        
- **Agent (–ê–≥–µ–Ω—Ç):**¬†–≠—Ç–æ —Å–∏—Å—Ç–µ–º–∞, –≥–¥–µ LLM –±–µ—Ä–µ—Ç –Ω–∞ —Å–µ–±—è —Ä–æ–ª—å "Reasoning Engine". –ê–≥–µ–Ω—Ç —Ä–∞–±–æ—Ç–∞–µ—Ç –≤ —Ü–∏–∫–ª–µ (Loop). –ù–∞ –∫–∞–∂–¥–æ–º —à–∞–≥–µ –æ–Ω –æ—Ü–µ–Ω–∏–≤–∞–µ—Ç —Ç–µ–∫—É—â–µ–µ —Å–æ—Å—Ç–æ—è–Ω–∏–µ –∏ –ø—Ä–∏–Ω–∏–º–∞–µ—Ç —Ä–µ—à–µ–Ω–∏–µ: "–í—ã–∑–≤–∞—Ç—å –∏–Ω—Å—Ç—Ä—É–º–µ–Ω—Ç?", "–ó–∞–¥–∞—Ç—å –≤–æ–ø—Ä–æ—Å –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—é?" –∏–ª–∏ "–ó–∞–≤–µ—Ä—à–∏—Ç—å —Ä–∞–±–æ—Ç—É?".
    
    - _–ö–ª—é—á–µ–≤–æ–µ –æ—Ç–ª–∏—á–∏–µ:_¬†–ò–Ω–≤–µ—Ä—Å–∏—è –∫–æ–Ω—Ç—Ä–æ–ª—è. –í Chain –≤—ã —É–ø—Ä–∞–≤–ª—è–µ—Ç–µ –ø–æ—Ç–æ–∫–æ–º, –≤ Agent –ø–æ—Ç–æ–∫–æ–º —É–ø—Ä–∞–≤–ª—è–µ—Ç –º–æ–¥–µ–ª—å.
        

### 1.2. –ö–æ–≥–Ω–∏—Ç–∏–≤–Ω—ã–µ –ê—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä—ã

–ê–≥–µ–Ω—Ç ‚Äî —ç—Ç–æ –Ω–µ –ø—Ä–æ—Å—Ç–æ —Ü–∏–∫–ª. –≠—Ç–æ "–º–æ–∑–≥", —Ä–∞–±–æ—Ç–∞—é—â–∏–π –ø–æ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω–æ–º—É –∞–ª–≥–æ—Ä–∏—Ç–º—É.

#### A. ReAct (Reason + Act)

–°–∞–º—ã–π –ø–æ–ø—É–ª—è—Ä–Ω—ã–π –ø–∞—Ç—Ç–µ—Ä–Ω (Yao et al., 2022).

1. **Thought (–ú—ã—Å–ª—å):**¬†–ê–≥–µ–Ω—Ç –∞–Ω–∞–ª–∏–∑–∏—Ä—É–µ—Ç –∑–∞–ø—Ä–æ—Å.¬†_"–ü–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å –ø—Ä–æ—Å–∏—Ç –ø–æ–≥–æ–¥—É –≤ –ü–µ–Ω–∑–µ. –ú–Ω–µ –Ω—É–∂–µ–Ω –∏–Ω—Å—Ç—Ä—É–º–µ–Ω—Ç¬†`get_weather`."_
    
2. **Action (–î–µ–π—Å—Ç–≤–∏–µ):**¬†–ê–≥–µ–Ω—Ç —Ñ–æ—Ä–º–∏—Ä—É–µ—Ç –≤—ã–∑–æ–≤ –∏–Ω—Å—Ç—Ä—É–º–µ–Ω—Ç–∞.¬†`get_weather(city="Penza")`.
    
3. **Observation (–ù–∞–±–ª—é–¥–µ–Ω–∏–µ):**¬†–°—Ä–µ–¥–∞ –≤—ã–ø–æ–ª–Ω—è–µ—Ç –∫–æ–¥ –∏ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç —Ä–µ–∑—É–ª—å—Ç–∞—Ç.¬†`Temperature: -15C`.
    
4. **Repeat:**¬†–ê–≥–µ–Ω—Ç –ø–æ–ª—É—á–∞–µ—Ç —Ä–µ–∑—É–ª—å—Ç–∞—Ç –∏ —Å–Ω–æ–≤–∞ –¥—É–º–∞–µ—Ç.¬†_"–î–∞–Ω–Ω—ã—Ö –¥–æ—Å—Ç–∞—Ç–æ—á–Ω–æ. –û—Ç–≤–µ—Ç: –í –ü–µ–Ω–∑–µ -15 –≥—Ä–∞–¥—É—Å–æ–≤."_
    

#### B. Plan-and-Execute (Plan-and-Solve)

ReAct —Å—Ç—Ä–∞–¥–∞–µ—Ç "–±–ª–∏–∑–æ—Ä—É–∫–æ—Å—Ç—å—é" ‚Äî –æ–Ω –≤–∏–¥–∏—Ç —Ç–æ–ª—å–∫–æ —Å–ª–µ–¥—É—é—â–∏–π —à–∞–≥. –î–ª—è —Å–ª–æ–∂–Ω—ã—Ö –∑–∞–¥–∞—á (–Ω–∞–ø–∏—Å–∞–Ω–∏–µ –∫–Ω–∏–≥–∏, –∞–Ω–∞–ª–∏–∑ –±–æ–ª—å—à–∏—Ö –¥–∞–Ω–Ω—ã—Ö) –Ω—É–∂–µ–Ω –ø–ª–∞–Ω.

1. **Planner:**¬†–û—Ç–¥–µ–ª—å–Ω–∞—è LLM –≥–µ–Ω–µ—Ä–∏—Ä—É–µ—Ç —Å–ø–∏—Å–æ–∫ —à–∞–≥–æ–≤ (1. –ù–∞–π—Ç–∏ –¥–∞–Ω–Ω—ã–µ, 2. –ü—Ä–æ–∞–Ω–∞–ª–∏–∑–∏—Ä–æ–≤–∞—Ç—å, 3. –ù–∞–ø–∏—Å–∞—Ç—å –æ—Ç—á–µ—Ç).
    
2. **Executor:**¬†–ê–≥–µ–Ω—Ç (–∏–ª–∏ Worker) –±–µ—Ä–µ—Ç —à–∞–≥ ‚Ññ1, –≤—ã–ø–æ–ª–Ω—è–µ—Ç –µ–≥–æ, –æ–±–Ω–æ–≤–ª—è–µ—Ç —Å–æ—Å—Ç–æ—è–Ω–∏–µ.
    
3. **Replanner:**¬†(–û–ø—Ü–∏–æ–Ω–∞–ª—å–Ω–æ) –ü–æ—Å–ª–µ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è —à–∞–≥–∞ –ø–ª–∞–Ω –º–æ–∂–µ—Ç –±—ã—Ç—å —Å–∫–æ—Ä—Ä–µ–∫—Ç–∏—Ä–æ–≤–∞–Ω.
    

### 1.3. Tool Use (Function Calling)

–ö–∞–∫ LLM –≤–∑–∞–∏–º–æ–¥–µ–π—Å—Ç–≤—É–µ—Ç —Å –≤–Ω–µ—à–Ω–∏–º –º–∏—Ä–æ–º?  
–†–∞–Ω—å—à–µ (2022-2023) –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–ª–∏ MRKL –∏ –ø–∞—Ä—Å–∏–Ω–≥ —Ä–µ–≥—É–ª—è—Ä–∫–∞–º–∏ (`Action: Search\nInput: Penza`). –°–µ–π—á–∞—Å —Å—Ç–∞–Ω–¥–∞—Ä—Ç–æ–º —Å—Ç–∞–ª¬†**Native Function Calling**¬†(OpenAI, Anthropic).

1. **Schema:**¬†–í—ã –æ–ø–∏—Å—ã–≤–∞–µ—Ç–µ –∏–Ω—Å—Ç—Ä—É–º–µ–Ω—Ç—ã –≤ —Ñ–æ—Ä–º–∞—Ç–µ JSON Schema (–∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∏ –∏–∑ Pydantic –º–æ–¥–µ–ª–µ–π).
    
2. **Binding:**¬†–í—ã "–±–∏–Ω–¥–∏—Ç–µ" (–ø—Ä–∏–≤—è–∑—ã–≤–∞–µ—Ç–µ) —ç—Ç–∏ —Å—Ö–µ–º—ã –∫ –º–æ–¥–µ–ª–∏.¬†`llm.bind_tools([weather_tool])`.
    
3. **Response:**¬†–ú–æ–¥–µ–ª—å –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –Ω–µ —Ç–µ–∫—Å—Ç, –∞ —Å—Ç—Ä—É–∫—Ç—É—Ä—É¬†`tool_calls`¬†(ID –∏–Ω—Å—Ç—Ä—É–º–µ–Ω—Ç–∞, –∞—Ä–≥—É–º–µ–Ω—Ç—ã).
    
4. **Execution:**¬†–í–∞—à Runtime (LangGraph) –≤—ã–ø–æ–ª–Ω—è–µ—Ç —Ñ—É–Ω–∫—Ü–∏—é –∏ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç —Ä–µ–∑—É–ª—å—Ç–∞—Ç —Å —Ç–µ–º –∂–µ¬†`tool_call_id`.

## 2. LangGraph: –ê—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä–∞ –∏ –ø—Ä–∏–º–∏—Ç–∏–≤—ã

LangGraph —Ä–µ—à–∞–µ—Ç –≥–ª–∞–≤–Ω—É—é –ø—Ä–æ–±–ª–µ–º—É LangChain ‚Äî –æ—Ç—Å—É—Ç—Å—Ç–≤–∏–µ –Ω–æ—Ä–º–∞–ª—å–Ω–æ–≥–æ —É–ø—Ä–∞–≤–ª–µ–Ω–∏—è —Å–æ—Å—Ç–æ—è–Ω–∏–µ–º (State Management) –∏ —Å–ª–æ–∂–Ω–æ—Å—Ç—å —Å–æ–∑–¥–∞–Ω–∏—è —Ü–∏–∫–ª–æ–≤.

### 2.1. State (–ì—Ä–∞—Ñ –°–æ—Å—Ç–æ—è–Ω–∏–π)

–í —Ü–µ–Ω—Ç—Ä–µ –≤—Å–µ–≥–æ ‚Äî¬†**State Schema**. –≠—Ç–æ¬†`TypedDict`¬†–∏–ª–∏¬†`Pydantic`¬†–º–æ–¥–µ–ª—å, –æ–ø–∏—Å—ã–≤–∞—é—â–∞—è "–ø–∞–º—è—Ç—å" –≤–∞—à–µ–≥–æ –≥—Ä–∞—Ñ–∞.

```python
from typing import TypedDict, Annotated

    # –°–∞–º–æ–µ –≤–∞–∂–Ω–æ–µ –ø–æ–ª–µ. –•—Ä–∞–Ω–∏—Ç –∏—Å—Ç–æ—Ä–∏—é —á–∞—Ç–∞.

    # –ù–ï –ø–µ—Ä–µ–∑–∞–ø–∏—Å—ã–≤–∞–π —Å–ø–∏—Å–æ–∫, –∞ –¥–æ–±–∞–≤–ª—è–π (append) –∏—Ö –≤ –∫–æ–Ω–µ—Ü".

    # –ö–∞—Å—Ç–æ–º–Ω—ã–µ –ø–æ–ª—è (–ø–µ—Ä–µ–∑–∞–ø–∏—Å—ã–≤–∞—é—Ç—Å—è –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é)
    user_id: str
    steps_count: int
```

**Reducer (`add_messages`):**¬†–≠—Ç–æ —Ñ—É–Ω–∫—Ü–∏—è, –æ–ø—Ä–µ–¥–µ–ª—è—é—â–∞—è, –∫–∞–∫ –æ–±—ä–µ–¥–∏–Ω—è—Ç—å —Å—Ç–∞—Ä–æ–µ –∏ –Ω–æ–≤–æ–µ —Å–æ—Å—Ç–æ—è–Ω–∏–µ.¬†`add_messages`¬†‚Äî —ç—Ç–æ –≤—Å—Ç—Ä–æ–µ–Ω–Ω—ã–π —Ä–µ–¥—å—é—Å–µ—Ä, –∫–æ—Ç–æ—Ä—ã–π —É–º–µ–µ—Ç:

- –î–æ–±–∞–≤–ª—è—Ç—å –Ω–æ–≤—ã–µ —Å–æ–æ–±—â–µ–Ω–∏—è.
    
- –û–±–Ω–æ–≤–ª—è—Ç—å —Å—Ç–∞—Ä—ã–µ —Å–æ–æ–±—â–µ–Ω–∏—è (–µ—Å–ª–∏ —É –Ω–∏—Ö —Å–æ–≤–ø–∞–¥–∞–µ—Ç¬†`id`). –≠—Ç–æ –∫—Ä–∏—Ç–∏—á–Ω–æ –¥–ª—è –∏—Å–ø—Ä–∞–≤–ª–µ–Ω–∏—è –æ—à–∏–±–æ–∫ –∏–ª–∏ Human-in-the-loop —Ä–µ–¥–∞–∫—Ç–∏—Ä–æ–≤–∞–Ω–∏—è.
    

### 2.2. Nodes (–£–∑–ª—ã)

–£–∑–ª—ã ‚Äî —ç—Ç–æ —Ñ—É–Ω–∫—Ü–∏–∏ Python. –û–Ω–∏:

1. –ü—Ä–∏–Ω–∏–º–∞—é—Ç —Ç–µ–∫—É—â–∏–π¬†`State`.
    
2. –í—ã–ø–æ–ª–Ω—è—é—Ç —Ä–∞–±–æ—Ç—É (–≤—ã–∑–æ–≤ LLM, –≤—ã–∑–æ–≤ API, –ø–æ–∏—Å–∫ –≤ –ë–î).
    
3. –í–æ–∑–≤—Ä–∞—â–∞—é—Ç**–æ–±–Ω–æ–≤–ª–µ–Ω–∏–µ** (Update) —Å–æ—Å—Ç–æ—è–Ω–∏—è.

    # –õ–æ–≥–∏–∫–∞: –≤—ã–∑—ã–≤–∞–µ–º –º–æ–¥–µ–ª—å —Å –∏–Ω—Å—Ç—Ä—É–º–µ–Ω—Ç–∞–º–∏

    # –í–æ–∑–≤—Ä–∞—â–∞–µ–º –¥–µ–ª—å—Ç—É. Reducer —Å–∞–º –¥–æ–±–∞–≤–∏—Ç —ç—Ç–æ –≤ –æ–±—â–∏–π —Å–ø–∏—Å–æ–∫.

### 2.3. Edges (–†–µ–±—Ä–∞)

    - –§—É–Ω–∫—Ü–∏—è –º–∞—Ä—à—Ä—É—Ç–∏–∑–∞—Ü–∏–∏ (`route_function`) —Å–º–æ—Ç—Ä–∏—Ç –Ω–∞ —Å—Ç–µ–π—Ç (–Ω–∞–ø—Ä–∏–º–µ—Ä, "–µ—Å—Ç—å –ª–∏ –≤ –ø–æ—Å–ª–µ–¥–Ω–µ–º —Å–æ–æ–±—â–µ–Ω–∏–∏¬†`tool_calls`?") –∏ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –∏–º—è —Å–ª–µ–¥—É—é—â–µ–≥–æ —É–∑–ª–∞.

    - _JSONB:_¬†–°—Ç–µ–π—Ç —Ö—Ä–∞–Ω–∏—Ç—Å—è –≤ JSONB, —á—Ç–æ –ø–æ–∑–≤–æ–ª—è–µ—Ç –¥–µ–ª–∞—Ç—å –∑–∞–ø—Ä–æ—Å—ã –∫ –∏—Å—Ç–æ—Ä–∏–∏ (–Ω–∞–ø—Ä–∏–º–µ—Ä, "–Ω–∞–π—Ç–∏ –≤—Å–µ—Ö –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª–µ–π, –∫—Ç–æ —Å–ø—Ä–∞—à–∏–≤–∞–ª –ø—Ä–æ —Ü–µ–Ω—ã").

## 3. –ü—Ä–æ–¥–≤–∏–Ω—É—Ç—ã–µ –ø–∞—Ç—Ç–µ—Ä–Ω—ã

### 3.1. Human-in-the-Loop (HITL)

–ò–ò –º–æ–∂–µ—Ç –æ—à–∏–±–∞—Ç—å—Å—è. –ò–Ω–æ–≥–¥–∞ –Ω—É–∂–µ–Ω —á–µ–ª–æ–≤–µ–∫, —á—Ç–æ–±—ã –ø–æ–¥—Ç–≤–µ—Ä–¥–∏—Ç—å –¥–µ–π—Å—Ç–≤–∏–µ (Approval) –∏–ª–∏ –∏—Å–ø—Ä–∞–≤–∏—Ç—å –¥–∞–Ω–Ω—ã–µ (Edit).

#### A. Breakpoints (–¢–æ—á–∫–∏ –æ—Å—Ç–∞–Ω–æ–≤–∞)

–ú—ã –º–æ–∂–µ–º —Å–∫–∞–∑–∞—Ç—å –≥—Ä–∞—Ñ—É: "–û—Å—Ç–∞–Ω–æ–≤–∏—Å—å –ø–µ—Ä–µ–¥ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏–µ–º —É–∑–ª–∞¬†`execute_payment`".

graph = builder.compile(interrupt_before=["execute_payment"])

1. –ê–≥–µ–Ω—Ç –¥–æ—Ö–æ–¥–∏—Ç –¥–æ —É–∑–ª–∞ –∏ –≤—Å—Ç–∞–µ—Ç –Ω–∞ –ø–∞—É–∑—É.

#### B. Time Travel & State Editing
